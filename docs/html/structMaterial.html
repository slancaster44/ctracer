Detailed Description

Represents a shape's material These values can be used by a Shader function to generate a color for a intersection

Member Function Documentation

◆ NewMaterial()

Material NewMaterial ( Tuple3  color)

Generates a solid material of a given color. The other material fields are filled in with sensiable defaults, and can be manipulated as desired

Parameters
'Tuple3color' The color of the generated color
Returns
'Material' the generated material

Default Values

  • Material.pattern = NewSolidPattern(color);
  • Material.ambient_reflection = 0.1;
  • Material.diffuse_reflection = 0.9;

  • Material.general_reflection = 0.0;
